Pages to design:
Admin Console
Select Hunt (or Create New Hunt)
Clue Hunt Settings (Enable Variant)
Stops & Media
Upload Image/Video for Stop
Add Clue/Hint Text

Media upload previews per stop
Code field per stop
Drag-and-drop stop ordering
